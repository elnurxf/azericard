Framework agnostic PHP package to proceed AzeriCard payments and reversal/clearance procedures.

## Install

Via Composer

``` bash
$ composer require elnurxf/azericard dev-master
```

## Usage

Please find examples at ```examples/``` folder

## Testing

``` bash
$ phpunit
```
